# coolzzin-test
내가 설마 쿨찐? 간단한 테스트를 통해 알아보는 웹사이트입니다.

## 1. 목표와 기능
### 1.1 목표
- Bootstrap의 활용
- 웹사이트의 배포 및 DNS 구매를 통한 도메인 연결
- SNS 공유 기능, 배너 광고 넣기
- SEO 최적화 과정 이해

### 1.2 기능
- 주어진 질문에 본인의 성향대로 답변을 하면 쿨찐인지 아닌지 결과를 알려줍니다.
- 각종 SNS로 공유할 수 있습니다.

## 2. 개발 환경 및 배포 URL
### 2.1 개발 환경
- 서비스 배포 환경
  - Netlify
### 2.2 배포 URL
- https://test-coolzzin.netlify.app/
